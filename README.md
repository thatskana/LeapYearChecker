# Leap Year Checker📆
A simple JavaScript program that finds and displays all leap years between year 2000 and 3000.

## How It Works

A year is considered a leap year when:

- It is divisible by 4 but not divisible by 100, or
- It is divisible by 400.

The program loops through every year from 2000 to 3000 and prints only the leap years.
